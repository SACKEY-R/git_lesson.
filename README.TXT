INTRODUCTION TO GIT/GITHUB FOR DATA SCIENCE

1.	What is Version Control?
Version Control is a system that records changes made to files over time so you can:
•	Track what changed
•	Know who changed it
•	Return to a previous version if something breaks
•	Work with many people on the same project safely
2.	What is Git?
Git is a Distributed Version Control System (DVCS) used to track changes in source code during software development.
•	Created by Linus Torvalds in 2005
•	Works offline
•	Tracks every change made to files
•	Allows multiple developers to work on the same project
